#### 符号表
符号表 std::vector（尽管是栈式，但由于有检索操作，设计成vector）
符号表记录
1. 变量名
2. 类型

块索引表 std::stack
1. 符号表记录指针（下标）
由于不允许函数嵌套定义，块索引表最多只有两个记录。

符号表操作
1. 插入：先调用检索操作判断是否重名，若否，则将符号插入
2. 检索：根据名字从后向前查找
3. 定位：进入函数/过程后定义第一个变量前执行定位操作，将符号表栈顶top（下标，即size）记录到块索引表。
4. 重定位：退出函数/过程后用块索引表top替换符号表的top（清除符号表从索引块表top至末尾的所有记录）
